# MusicStream

This project is an attempt to make a streaming music service.

## Technologies

### Backend

* Spring Boot
* Hibernate and Jpa as ORM (MySQL) and liquibase for the changelogs
* Lombok
* Mapstruct
* Junit and Mockito for the tests
